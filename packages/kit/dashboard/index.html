    About this data
    Calibrate records configured positions and bounded lifecycle signals. It does not record form values, clipboard contents, DOM text, URLs, query strings, hashes, or cookies.

    • State is in memory unless JSONL persistence is configured.
    • The collector is single-process.
    • Restarting clears state unless persistence is enabled.
